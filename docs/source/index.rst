OLAV documentation
===================

Welcome to the documentation for OLAV (or, equivalently, OLAV), the Off-road
Light Autonomous Vehicle. OLAV has been running autonomously since 2014 as an
off-road autonomous driving development platform at the Norwegian Defence
Research Establishment (FFI).

What is OLAV?
--------------

FFI open-sourced the OLAV codebase in 2023 after Aarhus University acquired the
vehicle. The original interfaces the underwent a major rewrite including the
migration from ROS to ROS 2.
